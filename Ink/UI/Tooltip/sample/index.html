Tooltip - demo

You can configure the text

  • data-tip-text="something"

Several tooltip positions:

  • data-tip-where="up"
  • data-tip-where="down"
  • data-tip-where="left"
  • data-tip-where="right"
  • data-tip-where="mousemove"
  • data-tip-where="mousefix"

Different colors

  • [no color defined]
  • data-tip-color="red"
  • data-tip-color="orange"
  • data-tip-color="blue"
  • data-tip-color="green"
  • data-tip-color="black"

Custom templates:

A random tooltip template

Write tooltip text here in this span, please

  • data-tip-template="#tooltip-template" data-tip-templatefield="span" data-tip-text="Something I wish to write in that span"

It totally works on scrolled content

The Spanish Inquisition

Biggles: Er.... Nobody...um....

Ximinez: Expects...

Biggles: Expects... Nobody expects the...um...the Spanish...um...

Ximinez: Inquisition.

Biggles: I know, I know! Nobody expects the Spanish Inquisition.

You can specify whether you want it to fade in and out, and how much time (in seconds) you want it to take.

  • data-tip-fade="0"
  • data-tip-fade="0.2"
  • data-tip-fade="0.5"
  • data-tip-fade="1"
  • data-tip-fade="10"
