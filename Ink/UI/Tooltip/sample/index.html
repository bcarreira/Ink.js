Tooltip - demo

You can configure the text

  • data-tip-text="something"

Several tooltip positions:

  • data-tip-where="up"
  • data-tip-where="down"
  • data-tip-where="left"
  • data-tip-where="right"
  • data-tip-where="mousemove"
  • data-tip-where="mousefix"

Different colors

  • [no color defined]
  • data-tip-color="red"
  • data-tip-color="orange"
  • data-tip-color="blue"
  • data-tip-color="green"
  • data-tip-color="black"

Custom templates:

  • data-tip-template="#tooltip-template" data-tip-templatefield="span" data-tip-text="Something I wish to write in that span"
